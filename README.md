# MindMates_Protothon
MindMates is an AI-powered emergency response web platform that enables rapid accident reporting, real-time ambulance and hospital coordination, first-aid guidance, blood group sharing, and emergency contact alerts to reduce response time and improve survival during road accidents.
# Problem Statement
Road accident victims often lose critical time due to:
- Late or incomplete accident reporting.
- Ambulances dispatched without case details.
- Hospitals unprepared before patient arrival.
- Families unaware during emergencies.
- No unified coordination system.
- Even small delays can significantly reduce survival chances.
# Solution

MindMates creates a centralized, real-time coordination system that:
1. Collects accident data instantly
2. Uses AI to assess severity
3. Dispatches ambulances intelligently
4. Notifies hospitals in advance
5. Alerts emergency contacts
This transforms emergency response from delayed reaction to proactive coordination.

# Key Features
- Rapid Accident Reporting
  - Description + image upload
  - Live GPS capture
  - Anonymous reporting support

- AI Emergency Analysis
  - Severity detection
  - First-aid guidance generation
  - Structured emergency summary

- Smart Ambulance Dispatch
  - Nearest ambulance notification
  - Navigation integration
  - Real-time updates

- Hospital Preparedness Dashboard
  - Receives case details before arrival
  - Displays blood group (if registered)
  - Enables ICU & equipment preparation

- Emergency Contact Alerts
  - Automatic notifications
  - Real-time location sharing
  - Continuous updates

- Real-Time Coordination
  - Witness → Ambulance → Hospital → Family
  - Centralized communication flow
